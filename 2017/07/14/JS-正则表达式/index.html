JS-正则表达式

发表于 2017-07-14

语法

  • \d可以匹配一个数字，等价于[0-9]
  • \w可以匹配一个单字字符 （字母、数字或者下划线） ，等价于[A-Za-z0-9_]
  • \W匹配一个非单字字符，等价于[^A-Za-z0-9_]
  • .可以匹配一个任意字符 ，换行符除外
  • \s匹配一个空白字符，包括空格、制表符、换页符和换行符
  • \S匹配一个非空白字符
  • *表示任意个字符，等价于 {0,}
  • +表示至少一个字符，等价于 {1,}
  • ?表示0个或1个字符，等价于 {0,1}
  • {n}表示n个字符
  • {n,m}表示n-m个字符
  • \转义
  • (x)匹配 'x'并且记住匹配项
  • [xyz]一个字符集合
  • [^xyz]一个反向字符集
  • A|B可以匹配A或B
  • ^表示行的开头或非的意思
  • $表示行的结束
